# PDFoutline

Use [pdf-lib](https://github.com/Hopding/pdf-lib) to add hierarchy outlines. Most of codes are from [issue 127](https://github.com/Hopding/pdf-lib/issues/127), 
hierarchy outlines are added by refering the [PDF specification](https://www.adobe.com/content/dam/acom/en/devnet/pdf/pdfs/PDF32000_2008.pdf) in pdf-lib.
